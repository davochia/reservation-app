# Neagen eservation system

## Tools

1. Java (Spring boot)
2. Swagger
3. Vue
4. Skywalking

### Start

To Start, simply run **_docker-compose up_** and visit port http://hostIp:8080 for the UI and http://hostIp:8088 for skywalking monitoring Tool
